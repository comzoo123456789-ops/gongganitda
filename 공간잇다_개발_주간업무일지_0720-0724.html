  • 관리자 회원 상세 요율 편집 모달 + '수수료율' 전용 탭(오름/내림차순 정렬)

  • 항목별 부분 환불 + 이용일 경과 건 정산 회수(클로백) 처리, 환불 시 정산 자동 제외

공간 대관 취소 수수료율 (D-day 차등)신규
  • 이용일까지 남은 기간별 표준 5구간(7일 100% · 5~6일 70% · 3~4일 50% · 1~2일 30% · 당일 0%)
  • 부대서비스는 리드타임 반영 별도 정책(7일 100% · 3~6일 50% · 2일 이내 0%)
  • 환불 모달에서 환불액·위약금 자동 계산, 위약금은 정상 수수료 차감 후 공급자 정산 — Node 21건 검증 통과
결제·환불 약관 정밀 검토 + 개정안 문서문서
  • 약관 4종(취소·환불/주문·결제/PG/파트너)과 실제 시스템 동작 정합성 교차 검증
  • 불일치·법적 보완점(청약철회·위약금 귀속·통신판매중개자 책임 등) 도출 + 현행 vs 개정안 비교 문서(PDF·Word·HTML)
정산 화면 개편 + KPI 카드 디자인 통일개선
  • 호스트·업체 정산 내역을 거래액·수수료·최종 입금액·결제상태가 한눈에 보이는 카드로 개편(상세 모달 제거, 영수증·국세청 버튼 상단 배치)
  • 날짜순·금액순·결제상태순 정렬 + 기간 지정 검색 + 합계 요약 바
  • 대시보드·정산·관리자 KPI 카드를 플랫 스트립으로 통일(회원용 페이지 제외)
  • 요약 카드 클릭 시 목록 상태값까지 즉시 필터(C-8), 알림 센터 슬라이드 애니메이션(C-9)
호스트 마이페이지 고도화 (A)신규

  • 내 등록 공간 노출/숨김 토글 스위치 — 카드 우측 상단 즉시 조작(A-2)
  • 공간 견적요청 3단계 파이프라인(신규→입찰→낙찰) 카드(A-3)
  • 정산 매출 리포트 차트(공간 단독 vs 패키지 비중) + 세금계산서 PDF 다운로드(A-4)
맞춤업체 마이페이지 고도화 (B)신규
  • 확정 건별 현장 4단계 상태(준비→세팅→진행→회수) — 게스트·호스트 실시간 알림 공유(B-1)
  • 1일 최대 처리 수량(CAPA) 설정 → 한도 초과 날짜 신규 견적 자동 차단(B-2)
  • 연동 현황 탭 신설 — 내 서비스 연동 공간의 호스트 숨김/차단 감지 + 참여 패키지 판매 현황(B-3)
